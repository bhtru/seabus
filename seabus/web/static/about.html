Technology

This app retrieves live marine positioning data broadcast from each seabus and displays it in real time. Position data is broadcast every 30-60 seconds while seabus is operating. Source for the app is here. This app is beta software, bug reports are very welcome! For more background here's a blog post with some technical details.
